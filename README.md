# Selenium Automation Framework

A production-grade UI automation framework built with Java, Selenium WebDriver, and TestNG.

## Tech Stack
- Java 17
- Selenium WebDriver 4.22.0
- TestNG 7.9.0
- Maven
- Log4j2
- Extent Reports 5.1.1
- Page Object Model (POM)

## Framework Features
- Page Object Model design pattern
- Data-driven testing with @DataProvider and CSV files
- Centralized configuration via config.properties
- Log4j2 logging to console and file
- Auto screenshot capture on test failure
- Extent Reports HTML reporting
- CI/CD integration with GitHub Actions
